## Brief Description - What This Code Sample Does

This is a full-stack Web3 Wallet Manager application that allows users to securely manage their cryptocurrency wallet addresses across multiple blockchains (Ethereum, Bitcoin, Solana, Polygon, etc.). 

Key features:
- JWT-based user authentication system (registration/login)
- RESTful API for wallet CRUD operations
- Multi-chain wallet management with labels and notes
- Search and filter functionality
- Blockchain explorer integration
- Responsive React frontend with modern UI components
- MongoDB for persistent data storage

The application demonstrates full-stack development skills, REST API design, authentication implementation, database integration, and clean code architecture suitable for production environments.

## What I Learned When Creating This Code Sample

Building this Web3 Wallet Manager taught me several valuable skills:

1. **Full-Stack Architecture**: I learned how to design and implement a complete full-stack application, connecting a React frontend with a FastAPI backend through RESTful APIs. This included handling CORS, state management, and ensuring seamless communication between layers.

2. **JWT Authentication**: I implemented secure user authentication using JWT tokens, including password hashing with bcrypt, token generation/validation, and protecting routes on both frontend and backend. This taught me about web security best practices.

3. **Async Python with FastAPI**: Working with FastAPI and Motor (async MongoDB driver) taught me about asynchronous programming in Python, which is crucial for building high-performance APIs that can handle multiple concurrent requests efficiently.

4. **MongoDB & NoSQL Design**: I learned how to design schemas for a NoSQL database, handle relationships between users and wallets, and perform efficient queries. This was different from SQL databases I'd worked with before.

5. **Modern React Patterns**: I implemented React hooks (useState, useEffect), React Router for navigation, protected routes, and component composition. I also learned to use Shadcn/ui components and Tailwind CSS for rapid UI development.

6. **Web3 Integration Concepts**: Understanding different blockchain ecosystems, wallet address formats, and integrating blockchain explorers gave me insights into Web3 infrastructure and how applications interact with blockchain networks.

7. **Code Organization**: I learned the importance of clean code structure, separating concerns (components, pages, utils), and writing maintainable code that others can easily understand and contribute to.

This project prepared me for real-world Web3 development by combining authentication, database operations, API design, and modern frontend frameworks - all essential skills for the MLH Fellowship.
